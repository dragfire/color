Make ASCII color strings.

## Usage:
```rust
use colci::Color;

Color::Cyan("All tests PASSED.").make();
Color::Yellow("Congratulations!").make();
```

## Supported Colors:
- Cyan
- Green
- Red
- Yellow
- Blue
- Magenta
